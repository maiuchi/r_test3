5 Snippets of dplyr and ggplot

To be honest, these two packages are the game changers. I use them almost everyday along with a couple of other packages.

5.1 What is dplyr?

dplyr is an R package that allows you to perform data manipulation easily.

5.2 Why is dplyr so handy?

  • Once you are out of your class assignments, you will need to perform data cleaning and manipulation by yourself to perform a data analysis.
    • Real world datasets are usually messy, unlike the dataset you get from your professors for your assignments (yes, they usually clean the data for you)
    • We say that data analysists spend about 70% of their project time on data cleaning/ preparations
  • The dplyr package helps you to perform data cleaning tasks

5.3 Filter/ subset data using dplyr

Let's say that you'd want to select only the May data from the airquality dataset we used in Chapter 4. You can do that pretty easily by using dplyr

  • Load the dplyr package
    • for the very first time, you need to install the package: install.packages("dplyr") or install.packages("tidyverse"")
  • Load the airquality dataset
  • Make sure the data structure
    • glimpse() is a dplyr fucntion to check data structures
  • Filter the dataframe by datapoints that are May
    • %>% is called piping. This means that grab from previous action/ item
      • In this case, it is telling to grab airquality dataset and then select the data that the "Month" column is 5
    • Be aware that we use double equal signs (==) to tell R that we need the value that is the same as 5 in the "Month" column
      • If you use a signle equal sign (=), R will think that you are asking to assign the value 5 to the "Month" column, which will give you an error message.
      • Single equal sign (=) is the same as (<-)
#load dplyr 
library(dplyr)

#load the dataset 
airquality

#check the data structure 
glimpse(airquality)

#filter May data from the dataframe 
airquality %>% filter(Month == 5)

#if you want to make a data frame called "may" which has only May data 
may <- airquality %>% filter(Month == 5)

5.4 Filter data with multiple conditions using dplyr

Now you know how to filter the data using dplyr. But what if you want to know how many days in May, 1973 were hotter than 74 degrees?

  • Load dplyr package
  • Load airquality dataset
  • Filter May and Temperature > 74
    • Notice that you use two pipes for this operation This code tells R, grab the airquality data and select May. And then from there, select the days that are hotter than 74
    • logical operators
    Operator Description
    < less than
    <= less than or equal to
    > greater than
    >= greater than or equal to
    == exactly equal to
    != not equal to 
#load dplyr 
library(dplyr)

#load the dataset 
airquality

#filter May and temp > 74 from the dataframe 
airquality %>% 
  filter(Month == 5) %>% 
  filter(Temp > 74)

5.5 What is ggplot?

ggplot is an R package that allows you to create nice-looking data visualization pieces

5.6 Why is ggplot so handy?

  • Data visualization is an important piece of your data analysis plan, because you need to tell your audience a story of what you found in a understandable and easily digestable manner
    • R can be your one-stop-shop for your data analysis project
      • ggplot will let you create graphs without using other software
  • You don't need to know design theory to make nice visualizations
    • ggplot's preloaded theme/ color schemes will help to create a nice visualization
  • If you are a design person like me, ggplot gives you to a freedom to customize your plots/ graphs
    • You can change font size, font color (using rbg or hex number), line types and more!
  • One you design a base "template" of your visual piece, you can reuse it over and over again by copy and pasting the code!
  • Good news for mappers like me, you can make maps using ggplot as well
